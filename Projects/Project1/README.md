## üß† Learning Outcomes

This project is a great exercise for beginners and covers fundamental concepts such as:

-   **DOM Selection:** Using `querySelector` and `querySelectorAll`.
-   **Event Handling:** Attaching and handling `click` events with `addEventListener`.
-   **DOM Manipulation:** Dynamically changing CSS properties of an element.
-   **`forEach` Loop:** Iterating over a `NodeList` to add event listeners efficiently.
