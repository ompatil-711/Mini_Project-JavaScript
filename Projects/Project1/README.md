Color Scheme Switcher
A simple and interactive web application that allows users to change the background color of the page by clicking on one of the four color options. This project is an excellent demonstration of basic DOM manipulation using JavaScript.

Features
Interactive UI: A clean and simple user interface.

Four Color Choices: Switch between Grey, White, Blue, and Yellow.

Instant Feedback: The background color changes instantly upon clicking a color box.

Lightweight: Built with vanilla HTML, CSS, and JavaScript with no external libraries.

Technologies Used
HTML5: For the basic structure and content of the web page.

CSS3: For styling the color boxes, text, and layout.
--   JavaScript: For handling click events and manipulating the DOM to change the background color.
